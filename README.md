Atom is a hackable text editor for the 21st century, built on [Electron](http://github.com/atom/atom-shell), and based on everything we love about our favorite editors. We designed it to be deeply customizable, but still approachable using the default configuration.

Visit [atom.io](https://atom.io) to learn more.

A shortcuts cheat sheet for Atom on Windows.
